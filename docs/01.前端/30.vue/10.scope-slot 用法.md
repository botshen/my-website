Vue 需要在插槽里获取组件的方法的时候可以使用 scope-slot 
# 定义：
```html
<slot name="content" :close="close"></slot>
//close是一个关闭的方法
```
# 使用：
```html
<template slot="content" slot-scope="{close}">
<g-button @click="close">关闭</g-button>
</template>
```
