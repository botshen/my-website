## 初始化项目

```bash
npm init -y 
// 锁定版本号
yarn add webpack@4.29.6 webpack-cli@3.2.3  -D
```
创建文件 webpack.config.js

require.js 发明了 amd 中文是异步模块定义
 
nodejs 模块定义
```js
module.exports = {}
```
commonjs 模块定义,只能在nodejs中使用

于是出现了统一的模块定义，umd，发现有 define 函数就用 amd，没有就用 commonjs
的 module.exports

## 打包项目

```bash
npx webpack
npx webpack-dev-server
```
